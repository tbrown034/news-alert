## Technical Notes

### Handle Format
- Telegram handles: `@username` or `t.me/username`
- API access: Use channel username without `@`
- Public channels don't require joining

### Posting Patterns
- **High-frequency** (30+ posts/day): DeepState, Abu Ali, Disclose.tv, Reuters
- **Medium-frequency** (10-30 posts/day): Most OSINT channels
- **Event-driven** (varies): Tzofar alerts, official military channels
- **Low-frequency** (5-10 posts/day): Think tank style (Amwaj, DD Geopolitics)

### Content Types
- **Maps**: DeepState, Rybar, Liveuamap
- **Translations**: War Translated, Abu Ali Express
- **Alerts**: Tzofar, IDF Official
- **Analysis**: Amwaj, DD Geopolitics, Colonel Cassad
- **Aggregation**: Intel Slava, Disclose.tv, OSINT TV

### Language Distribution
- **English-first**: DeepStateEN, War Translated, Abu Ali EN, Iran Intl EN
- **Ukrainian**: DeepStateUA, DIUkraine, Stratkom
- **Russian**: Rybar, Colonel Cassad, Astra, WarGonzo
- **Multi-language**: IDF Official (Hebrew/English/Arabic)

## Validation Checklist

Before adding a channel, verify:

- [ ] Handle exists and is public
- [ ] Active in last 7 days
- [ ] Posts relevant content (not just memes/ads)
- [ ] Subscriber count > 10K (for quality signal)
- [ ] Not on blocklist
- [ ] Content type matches our use case
